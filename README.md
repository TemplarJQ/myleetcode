附一个我印象深刻的东西：\
```
public static int part(int[] arr, int low, int high) {
                int temp = arr[low];
                while(low<high) {
                        while(arr[high]>temp&&high>low) {
                                high--;
                        }
                        arr[low] = arr[high];
                        while(arr[low]<temp&&low<high) {
                                low++;
                        }
                        arr[high] = arr[low];
                }
                arr[high] = temp;
                return high;
        }

        public  static void quick_sort_go(int[] arr,int low,int high) {
                if(low>=high) {return ;}
                int mid = part(arr,low,high);
                quick_sort_go(arr, low, mid -1);
                quick_sort_go(arr, mid+1, high);
        }
```
15题三数之和的这个有点难度。

典型二分法：
```
int start = 0;
        int end = nums.length - 1;
        while(start<end){
            int mid = start + (end - start)/2;
            if(nums[mid]==target){
                return mid;
            }else if(nums[mid]<target){
                start = mid + 1;
            }else{
                end = mid - 1;
            }
        }
        if(nums[start]==target){
            return start;
        }else{
            return -1;
        }
```
